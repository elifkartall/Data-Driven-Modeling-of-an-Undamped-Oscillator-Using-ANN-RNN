#Overview

This project focuses on learning the dynamics of a nonlinear system (Undamped Oscillator 2) using data-driven approaches.

Two neural network architectures are implemented and compared:

Artificial Neural Network (ANN) – learns direct state mapping
Residual Neural Network (ResNet) – learns the difference between consecutive states

The dataset is generated numerically using a Runge-Kutta method under multiple initial conditions.

The main objective is to evaluate whether these models can successfully learn system dynamics and to compare their performance in terms of accuracy and stability.

#📂 Dataset

The dataset is synthetically generated by solving the dynamical system.

100 different initial conditions
80% training data
20% test data
Numerical solutions are obtained using a Runge-Kutta-based solver.

#📊 Results

The following results are obtained from the trained models:

ANN successfully captures system dynamics in short-term predictions
ResNet fails to maintain stability and diverges over time
ANN achieves faster and more stable convergence
